This project is a **SMIT (Saylani Mass IT Training) assignment** based on creating a multi-step form using HTML and CSS.

## Project Overview

This project contains a simple multi-step form with three sections:

1. Account Details
2. Personal Details
3. Review Section

The user can navigate through different steps and fill in the required information.

## Features

- Multi-step form layout
- Account details form
- Personal details form
- CNIC input boxes
- Review and feedback section
- Responsive design
- Simple and clean UI styling

## Technologies Used

- HTML5
- CSS3
